Crazy Ex Projects

Are you a programmer who starts multiple projects but never finishes them?
Do you feel like the ideas you had are worth it, and might need a second change?
If you answer yes to these questions, then this is for you.

Presenting CEP, it is a bot on telegram, which has only one job. To remind you of all the projects you have started, but have abandoned them ever since. In the Alpha stage, all CEP does is look for projects with last commit between 3-9 months ago, and messages you with Project name, description and link.

Let's Get Started

  1. Login with Github, github might send you a verification code to your email address
  2. Connect Telegram
  3. click on start
  4. Get message from us, every Friday.
